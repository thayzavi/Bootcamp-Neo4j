# Bootcamp-Neo4j

#DESAFIO 01
- Análise de Filmes com Neo4j Aura
  
Este projeto consiste na modelagem e implementação de um grafo de conhecimento para um serviço de filmes e séries, utilizando banco de dados orientado a grafos (Neo4j) e linguagem de consulta Cypher.

  O objetivo é representar usuários, filmes, séries, atores, diretores e gêneros, bem como os relacionamentos entre essas entidades, permitindo análises e futuras recomendações.

#DESAFIO 03
- Análise de Redes Sociais com Neo4j Aura

Este projeto consiste na criação de um banco de dados em grafo para uma plataforma de rede social, com o objetivo de analisar engajamento, interações entre usuários, popularidade de conteúdos e comunidades de interesse, utilizando o Neo4j Aura.

O sistema foi desenvolvido como um protótipo funcional, capaz de responder a consultas complexas, explorando os benefícios do modelo de grafos.
